Темы для командной работы по 1-3 человека

В зависимости от номера темы, смотрите код проекта. Его нужно доработать, подключив вашу БД. Для первых 5 тем подключение через php, для 6-10 через API. Стек любой

Задание на практику:
Разработать базу данных и меры её защиты, согласно индивидуальной теме. Провести внедрение БД в готовый учебный продукт.

Задачи:
1. Изучить предметную область, составить список вопросов заказчику. Получить информацию путем интервью с руководителем практики.
2. Составить ER-диаграмму 
3. Реализовать базу данных
4. Внедрить полученную базу данных в веб-приложение
5. Провести тестирование БД на безопасность, в том числе на возможность sql-инъекций и несанкционированного доступа через API.
6. Составить резюме, в котором будет отражен результат выполненной работы как пет-проект или как полученные навыки
